# Online-Study-Resource-Hub

To clone and run this project, create two terminals one for *client* and another for *server*

## Clone project
```
git clone https://github.com/Hemanth-5/Online-Study-Resource-Hub.git
```

## Client setup
```
cd client
npm install
```

## Server setup
```
cd server
npm install
```

## Run
### Server
```
npm run dev
```
### Client
```
npm start
```
